## Attached images

{{imageCount}} image(s) are attached, numbered starting from 1. A short
description of each is provided in an "Attached images" block in the user message
— use those descriptions to decide where each image fits.

- Place each image where it most naturally belongs based on what it shows AND the
  surrounding text. Different images can land in completely different parts of the
  note; if an image illustrates a specific concept, put it directly under that
  section.
- Reference each image with the literal placeholder syntax `![short caption](image:N)`
  — where N is the image number (1-indexed). Do NOT invent your own URL.
- Every attached image must appear exactly once in the output, each on its own
  line.
- The caption should briefly describe what is in the image (you can base it on
  the provided description).
